      • What is HTML?

      HTML stands for Hypertext Markup Language, and it's the code used to structure web pages and their content. HTML is the foundation of the web and tells web browsers how to display text, images, and other media on a page.

      Features of HTML5

          Semantic Elements

          Canvas Elements
          Local Storage
          Responsive Images
