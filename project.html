Captcha Integrated Virtual Auction

The purpose of this project is to build a "Captcha Integrated Virtual Auction" a place for buyers and sellers to come together and trade almost anything. The objective of this application is to provide a facility for conducting online auction of products using the high security Captcha System. In this application Registered users can handle the products and Bid for the product. Customer can manually or automatically Bid the product with captcha facility. Registered Admin can monitor and approve the products and Bid requests.

Advantages

  • Less Time consuming
  • Data authentication by Individual persons
  • Data are easily accessible
  • Retrieval of data takes lesser time

Software Requirements

  • Front end : Java,JSP
  • Back end : My SQL
  • Operating System : Windoews XP, Windows 7, Windows 10
  • Software Tools : Eclipse
  • Server : Apache Tomcat
